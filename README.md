# Music Box Factory

Music Box Factory is a tool for algorithmic and programmatic creation of simple loops for emulation of music boxes.

## Feature Ideas

Here are some features that could be implemented to enhance Music Box Factory:

1. **MIDI Export Capabilities**: Allow exporting generated loops to standard MIDI files. This will make it possible to use the melodies in Digital Audio Workstations (DAWs) or export them for physical punch-card printing.
2. **Comb/Scale Constraint Enforcer**: A feature to map and constrain generated notes strictly to the available pitches (teeth) of specific physical music boxes (e.g., standard 15-note or 30-note diatonic/chromatic mechanisms).
3. **Built-in Algorithmic Generators**: Implement tools like Markov Chains, Cellular Automata, or L-Systems specifically tailored for music box patterns to easily generate pleasing and repetitive melodies.
